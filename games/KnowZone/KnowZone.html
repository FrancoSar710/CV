El juego consiste en recorrer un laberinto generado de forma procedural el cual hay que despejar sin que te mate el minotauro El juego fue programado con Python generando la API de cero. desde la ubicación de las imágenes hasta los movimientos todo fue programado sin el uso de un motor los dos algoritmos principales son el que genera el laberinto de forma procedural y el algoritmo de búsqueda A* el juego demoró en programarse 2 meses y medio y fue el primer proyecto relativamente grande en el que participe
